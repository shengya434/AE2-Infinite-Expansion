# AE2 Addon — Infinite Expansion v1.1.0

**Minecraft 1.20.1 · Forge · AE2 附属模组**

---

## 📖 简介

你是否曾对着 AE2 的存储限制叹气？是否幻想过一个元件装下整个宇宙？

**AE2 Infinite Expansion** 为你带来真正的无限存储体验。致敬某 MAX 合成存储器与某整合包的增强无限盘，但不止于此。

---

## ✨ 主要内容

### 万能无限存储元件
一块元件，三种模式，随心切换：

| 模式 | 名称 | 玩法 |
|------|------|------|
| 🟢 **Mode 1** | 无限制存储 | 无限容量 + 全 AE 类型，像普通盘一样存取 |
| 🟡 **Mode 2** | 自定义无限 | 阈值自动升无限 / 存入即无限 / 白名单臻藏，三种工作模式随意切换 |
| 🟣 **Mode 3** | 全类型无限 | 遍历注册表，所有物品流体存入即无限 |

**更厉害的是：**
- 内部 **BigInteger** 存储，突破 Java long 的 64 位天花板——显示上限 9.22E18，真实存储无限累加
- 模式切换自动刷新，无限物品保留不丢失
- 中文拼音搜索 + 分类过滤面板，几千种物品也能轻松管理
- JEI 拖拽物品到背包槽位直接加入白名单
- 分包网络传输，再多的类型也不会炸包

### 无限合成存储器
单个方块提供 **9.22E18 字节** 合成存储空间，万亿级合成永不爆仓。

### 无限并行处理单元
单个方块提供 **21.47 亿线程**，配合 Mixin 移除原版 16 线程上限——百万级子任务的合成也能秒级完成。

---

## ⚙️ 技术特性

- **WorldSavedData 持久化**：大量存储数据存到世界存档，ItemStack NBT 轻量化
- **GridMixin**：AE2 原生不支持子类方块注册——已修复
- **Tooltips 修复**：TB/PB/EB 级字节数正确显示，不再越界
- **溢出安全**：所有数值运算采用饱和算术，不崩不炸

---

## 📦 安装

1. 安装 Forge 47.x+
2. 安装 AE2 15.x+
3. （可选）安装 JEI 15.x+ 获得白名单拖拽功能
4. 将 jar 放入 `mods` 文件夹

**前置（必需）：** Forge ≥ 47, AE2 ≥ 15  
**前置（可选）：** JEI ≥ 15

