# AE2 Addon — Infinite Expansion ⚡

**Minecraft 1.20.1 · Forge · AE2 附属模组**

为 AE2 带来真正的无限存储体验！致敬某 MAX 合成存储器与某整合包的增强无限盘。

---

## 📦 内容

### ✦ 万能无限存储元件

| 模式 | 名称 | 说明 |
|------|------|------|
| **Mode 1** | 无限制存储 | 无限容量 + 全 AE 类型，正常存取 |
| **Mode 2** | 自定义无限 | 三种工作模式（阈值/存入无限/臻藏），白名单 + 通用阈值 |
| **Mode 3** | 全类型无限 | 遍历注册表，所有物品+流体存入即无限 |

**特性：**
- 内部使用 `BigInteger` 存储，突破 `Long.MAX_VALUE` 限制
- AE 终端显示上限为 `Long.MAX_VALUE`，真实存储持续累加
- 模式间切换自动刷新、无限物品保留
- 中文拼音搜索 + 分类过滤面板
- JEI 拖拽加入白名单
- 分包网络传输，支持数千种类型
- 世界存档级别数据持久化（`WorldSavedData`）

### ✦ 无限合成存储器
- 单个方块提供 **`Long.MAX_VALUE`** 字节合成存储空间
- 合成长整型溢出全周期支持

### ✦ 无限并行处理单元
- 单个方块提供 **`Integer.MAX_VALUE`** 线程
- 搭配 `CraftingCPUClusterMixin` 移除原版 16 线程上限
- 海量并行合成，万亿级配方秒级完成
